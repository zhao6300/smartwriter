# 智慧文案平台 - 智能公众号Agent工具

智慧文案平台 是一个基于前后端分离架构（B/S）的多用户智能推文创作 Agent 平台。系统围绕大型语言模型（LLMs）的特性进行深度开发，通过结构化的多分支工作流生成设计、独立容器项目化管理体系，大幅提升并保障了 AI 辅助长文案输出的逻辑严谨性与可用落地质量。

## ✨ 全新核心特性

- **全局项目工作台 (Dashboard & Workflow)**: 彻底抛弃单页面覆盖写作，提供“项目管理器”仪表盘。任何中途离开、断网或保存的分支动作，皆可实现**百分百断点续写恢复**。您的每一篇多阶段草稿都作为独立容器被完好保护。
- **神级多分支推步 (Triple-Outline Workflow)**: 颠覆传统的“黑盒一步成稿”。
  1. 向大模型下达立意要求。
  2. 底层并行推算出 **3 份风格迥异且带有详细共鸣剖析的结构化大纲**供您审查挑拣。
  3. 内置前端 IDE 级的 JSON 大纲微调面板，人工修正定方向后，开启流式(Server-Sent Events)文字注入。
- **跨平台现代交互 (Dark Mode & RWD)**: UI 全面应用高规格的 Glassmorphism 毛玻璃拟态风格与 Inter 商用字体。附带全局日夜间模式顺滑无缝切换（☀️/🌙），且页面底层响应式重写，手机至大视野 PC 皆有绝佳沉浸体验。
- **全网模型随意组装 (BYO-Config)**: 控制台开放了高度自由的 API 网关映射，从 OpenAI 到千问(Qwen)、DeepSeek 等协议兼容者均能任意拔插选用。
- **超级监控站 (Supervisor Portal)**: 内置具备高权隔离机制的管理中心 `/admin`。可一揽子总览全站沉淀的数据资产画像与每位作者的生产吞吐量。

## 🏗️ 架构与技术选型

为保持轻量便捷并易于扩展，系统配置为包含两个子 Workspace 的 Monorepo。

- **前端 (`src/client`)**: `React 18` + `Vite` 构建引擎。`React-Router-DOM` 与极简的 `Zustand` 集中接拉数据总线，纯野生 CSS Variables 驱动光影交错美学。
- **服务端 (`src/server`)**: `Node.js` + `Express`。数据层为强类型的 `Prisma ORM`，底层基座为原生 `SQLite` 零配置推移。

## 🚀 极速部署运行起步指南

### 前置环境需求
- `Node.js` (强烈推荐 v18 及以上版本)

### 步骤 1: 全栈一键依赖安装

在您的**工程根目录**下执行：
```bash
npm install
```

### 步骤 2: 初始化本地 SQLite 强类型数据库结构

进入服务端目录执行 Prisma 模型同步，并创建出 `dev.db` 本体：
```bash
cd src/server
npx prisma db push
```
*(注：如果由于系统执行策略限制找不到 npx 等报错指令，请尝试退出终端通过外部管理员权限或直接调用全局库唤醒该表册同步。)*

执行完毕后，回退至根节点：
```bash
cd ../../
```

### 步骤 3: 启动并行开发网络

根目录下一条命令并行拉起前后端服务站：
```bash
npm run dev
```

### 🎈 核心链路探索建议

启动后服务池将分别监听：`http://localhost:5173` (用户大门) 与 `http://localhost:3001` (底层运算)。
1. 请先在此大门注册一个名为 `admin` 的特殊账号！本系统硬编码了白名单提权后门，首注册的 `admin` 即会立刻升华签发 ROOT 最高权限包。
2. 转站到普通登录，即可进入创作者的平行宇宙创建您无尽的爆款草图。
3. 如果您想行使权利查验系统繁荣度，请在地址栏手动造访 `http://localhost:5173/admin/login` 输入 `admin` 密码，即可降维进入监察室！
