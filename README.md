# 智慧文案引擎 (AI Copywriting Agent Platform)

智慧文案引擎是一个基于前后端分离架构（B/S）的多用户智能辅助创作平台。本项目围绕大型语言模型（LLMs）的函数调用（Function Calling）与任务编排能力展开，将长文案创作转化为可控的工作流，提供结构化的大纲推演、段落级的精细调优以及灵活的外部工具扩展。

---

## 主要功能特性

### 1. 结构化多分支推演流程
- **并行大纲生成**：接收初始提示词后，系统由模型并行推断出 3 个不同思路的结构化大纲（JSON 抽象）。
- **人工介入反馈**：开发者或用户可在前端对渲染后的大纲细分节点进行人工修正、删减，也可以指定单一节点让模型针对性重写，确保大局逻辑可控。
- **流式注水生成**：大纲锁定后，系统通过 Server-Sent Events (SSE) 流式传输分段生成最终长文本，保障响应速度与上下文连贯。

### 2. 段落级内容重塑
- **微观重构**：文章成形后，支持在 Markdown 渲染视图下对某一段落进行局部选中。大模型会在保留全局上下文状态的同时，仅对该片段进行更新或重写，提升后期修改效率。

### 3. 可拔插的 Agent 扩展模块 (MCP 架构)
- **外部工具集成**：支持通过 Web/配置面板动态注册外部 API 或数据源（参考 Model Context Protocol 规范理念）。
- **内置搜索抓平台**：现默认预置了基于 DuckDuckGo 的网络搜索模块（SSR 爬取），在写作中途如判定需要事实材料，模型能自主开启查找。

### 4. 创作范式沉淀 (Template 引擎)
- 对已验证有效的行文结构和逻辑脉络，可逆向抽取保存为“模式模板（Template）”。以便在同类型主题进行创作时作为系统级 prompt 被隐式注入，实现业务规范复用。

### 5. 持久化数据与后台审计
- **独立容器级管理**：应用内所有的创作请求均落实为相对独立的 Project 实体。每次生成的文本阶段、大纲版本变更都能跨会话保存。
- **多用户及监控后台**：集成 JWT 鉴权和全局 Dashboard (`/admin`)。支持概览所有用户的 API 消耗、存储记录及活跃统计行为。

## 技术架构

采用 Monorepo 项目结构组织：

- **前端 (`src/client`)**: 
  - `React 18` + `Vite` 构建引擎。
  - `Zustand` 处理复杂状态流转与本地持久化。
  - 标准化 CSS Variables 实现浅色/深色 (Dark Mode) 无缝切换与响应式排版。
- **服务端 (`src/server`)**: 
  - `Node.js` + `Express`。
  - 通过 `Prisma ORM` 进行数据层映射，内置原生 `SQLite` 作为零配置底层存储方案。

## 本地开发指南

### 前置要求
- `Node.js` (建议 v18 LTS 及以上版本)

### 1. 安装依赖
在项目根目录下执行安装：
```bash
npm install
```

### 2. 数据库初始化
进入服务端目录，同步表结构并生成本地 `dev.db` 实体：
```bash
cd src/server
npx prisma db push
```

同步完成后退回根目录：
```bash
cd ../../
```

### 3. 本地启动
根目录并发启动前端打包与服务端监听：
```bash
npm run dev
```

## 使用提示

- 默认客户端地址：`http://localhost:5173`
- 默认服务端 API 基础路径： `http://localhost:3001`
- **后台管理系统进入方式**：请通过 `/admin/login` 手动进入后台入口。由于系统的初始化安全策略，**平台第一位注册用户只要取名为 `admin`** 即自动提升为 Root 最高权限。使用该账号才能访问完整的监控仪表板。
- 支持在前端工作台设置独立配置你的大语言模型 (API Base / API Key)，兼容符合标准 OpenAI SDK 输出的闭源或开源模型（如千问、DeepSeek等）。
