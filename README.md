# ✨ 智慧文案引擎 (AI Copywriting Agent Platform)

> **不只是一台打字机，而是你最贴心的全栈内容创作与运营中枢。**

**智慧文案引擎**是一个基于前后端分离架构（B/S）的多用户智能推文、长文内容创造工坊。它突破了传统大语言模型“一键黑盒输出”的简陋体验，将 AI 驱动的内容产出升级为**结构式推演、段落级精琢、动态知识库挂载**的“工作流 (Workflow)”范式。它在确保逻辑严密、风格可控的前提下，为你规模化地产出真正能吸引用户的优质爆款文案。

---

## 🌟 核心杀手锏特性

### 1. 🎭 拒绝平庸，神级「多分支推演」工作流
抛弃传统对话框那种“一步死板出稿”的废案流程！
你只需给出一个模糊的灵感，底层深核将并行演化出 **3 份风格迥异且带有详细分析的骨架大纲**。你可以在可视化的 JSON 白板上一键拾取段落结构并随意人工涂写，确认完美后，让大模型顺延骨架完成万字鸿篇的精准灌注。

### 2. 🧰 结对编程般的内容精修 (段落级微琢)
文章输出完毕并不等于工作结束！当你在阅读成品时，如果不满其中某一段落，可以直接选中该区块命令 AI：“把这段改得更幽默点”或“多给我加一个现实案例”。局部精准替换引擎会在知晓上下文的前提下进行完美融合修改，保留整体一致性。

### 3. 🧩 彻底打通外部大脑：MCP 节点池与插件架构
闭门造车是做不出时效性爆款的。控制台内建强大的 **工具中心 (Agent Tools)**：
- **🌐 原生集成全网智能检索 (DuckDuckGo)**: 创作时遇到盲区？模型可自主开启搜索引擎实时爬虫去确认最新资讯，融进最终文案里。
- **🔌 自由接入外部 MCP 私有协议（知识库/业务逻辑池）**：只需要进入配置页，在独立的图形界面中随意接入或拔除外界业务的配置节点数据，你的 Agent 瞬间可以连接全公司的企业智能库资源。

### 4. 📚 沉淀生产力：智能「套路」逆向萃取引擎
遇到爆款好文，直接丢给平台。引擎会彻底将文章剥皮抽筋，逆向拆解出底层吸引人的“套路与框架骨骼”，一键存入属于你的私人模板仓库。从此，创作就像搭积木一样高效复用成功经验。

### 5. 🛡️ 企业级多容器控制与跨平台美学体验
采用最前沿的 UI 技术堆栈（光效透传玻璃态、日夜间无边界自适应 `Dark Mode`、全终端响应适配），打造出让人上瘾的使用体验。更自带强大的 **`Supervisor 审计门禁`**，允许运营高频分析全局坐席的产出效率及账户运转负载健康。所有用户的长文本草稿也独享沙盒环境，拥有无敌容错级别的意外断点保存保护。

## 🏗️ 超轻量级，性能卓越的现代化架构栈

只为极客和前沿探索打造的清爽 Monorepo：
- **前端 (`src/client`)**: 响应式内核 `React 18` + `Vite`极速构建，配载无痛的 `Zustand` 极简流状态穿透。
- **服务端 (`src/server`)**: 纯粹稳固的 `Node.js` + `Express`。利用强类型 `Prisma ORM` 连接超极速原生嵌入式 `SQLite`，没有繁重的 Docker 锁链，跑通它正如呼吸一样自然。

## 🚀 极速部署起步，让 AI 立刻为你打工

### 前置环境需求
- 基础运行环境：`Node.js` (强烈推荐 v18 及以上版本)。

### 步骤 1: 全栈一键魔法依赖注入
在**工程根目录**下打开终端执行：
```bash
npm install
```

### 步骤 2: 唤醒并重置强类型脑核 (SQLite 模型建构)
进入服务端目录，推送实体模型至数据库表：
```bash
cd src/server
npx prisma db push
```
*(如果你的控制台报错找不到 npx 可执行命令，请检修本地 NodeJS 环境变量配置。)*

配置成功后返回起爆平台：
```bash
cd ../../
```

### 步骤 3: 引擎点火！启动双擎热更开发站
直接在系统根目录下并发拉升全部进程：
```bash
npm run dev
```
