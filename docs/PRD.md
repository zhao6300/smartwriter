# 智能公众号文案创作Agent平台 - 产品需求文档 (PRD)

## 1. 产品概述
### 1.1 产品背景
随着自媒体和公众号运营的日益普及，高质量内容的持续产出成为运营者的痛点。本项目旨在打造一个基于大语言模型（LLM）的智能文案创作Agent平台，通过结构化的“主题 -> 思考扩展 -> 大纲 -> 全文 -> 编辑”工作流，协助用户高效产出符合特定风格和逻辑的公众号推文。

### 1.2 目标用户
- 独立公众号主、个人自媒体创作者
- 企业新媒体运营团队
- 内容营销专员

## 2. 核心功能模块与业务流程
### 2.1 整体业务流程
1. **配置模型**：用户进入系统后，需配置自己的通义千问（Qwen）、OpenAI或其他任何兼容OpenAI API格式的模型参数。
2. **输入主题**：给定文案核心主题及基本诉求。
3. **Agent自动脑暴与大纲生成**：Agent根据主题进行深度“思考”，扩展相关背景信息，并生成结构化大纲（包括文章需要重点表达的点、建议的语言风格、逻辑讲解顺序等）。
4. **人工确认与调整**：用户在线检查大纲内容，并可手动增加/删除部分逻辑节点。
5. **生成全文**：Agent根据最终确认的大纲，撰写长篇图文并茂的文案。
6. **在线编辑与导出**：在富文本编辑器中进行精修和排版，完成定稿。

### 2.2 功能详情
#### 2.2.1 多账户体系
- **基础认证**：支持账号/密码本地注册与登录。
- **沙盒隔离**：不同用户的文案草稿、API配置数据完全相互独立隔离。

#### 2.2.2 AI模型自定义配置 (核心控制点)
- **参数配置**：
  - `API Base URL` (如: `https://api.openai.com/v1` 或 `https://dashscope.aliyuncs.com/compatible-mode/v1` 等)
  - `API Key` (用户私有的大模型调用凭证)
  - `Model Name` (如 `gpt-4o`, `qwen-max`, `qwen-plus`, `deepseek-chat` 等)
- **连通性测试**：提供一键发送Ping请求功能，验证用户的URL、Key及Model是否配置有效。

#### 2.2.3 核心创作工作流 - 第1阶段：大纲构建器
- **意图拆解**：
  - 用户只需输入核心词（例：“职场新人如何做向上管理”）。
- **Agent拓展策略**：
  - **背景与痛点拓展**：自动分析该主题下受众的核心焦虑点。
  - **核心表达点**：提炼3-5个不可或缺的干货论点。
  - **语言风格**：可由AI基于主题自动推荐（幽默风趣、专业严谨、毒舌扎心、情感共鸣），也可由用户覆盖修改。
  - **讲解顺序**：动态生成典型的爆款文章结构（如：现象引入 -> 原因深度剖析 -> 实操方法论 -> 价值观升华）。

#### 2.2.4 核心创作工作流 - 第2阶段：全文撰写与编辑器
- **上下文继承**：AI严格遵照【第1阶段】确定的大纲与基调风格生成内容。
- **在线富文本编辑器**：
  - 展示最终生成的Markdown及渲染效果。
  - **局部沉浸式AI编辑**：支持选中某段文本，执行“扩写”、“缩写”、“语气优化”、“换种说法”。
- **文档管理**：所有过程自动存储历史版本，防止内容丢失。

## 3. 非功能性与体验需求
- **流式交互 (Streaming)**：文案生成为降低等待焦虑，必须全链路支持打字机效果（SSE流式返回）。
- **数据安全**：用户的 `API Key` 不能明文留存在前端缓存，后端落库需加盐加密处理。
- **现代化极致UI**：
  - 采用极简设计语言，放弃传统后台的复杂表单感。
  - 大量运用悬浮、毛玻璃(Glassmorphism)效果、丝滑的骨架屏动画与过渡动画（Micro-animations）。
  - 给用户一种前沿高端Agent产品（类似Notion AI级别）的视觉 Wow 体验。
